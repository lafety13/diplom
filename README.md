## Minimal system requirements:

* PHP 5.6 or higher
* MySQL 5.5+
* Needed PHP modules
    * GD PHP Extension
    * PDO PHP Extension
    * INTL PHP Extension

## Installation and getting started

1. There are 2 ways of installation:
  * Use GitHub: simply download the zip
  * Use Git: `git clone https://github.com/lafety13/diplom.git`
2. ./init
3. ./yii migrate
